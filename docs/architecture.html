架构亮点

  • 基于 Hermes Agent 不重复造引擎
  • Skill 标准化封装，即插即用
  • DeepSeek V4 主力推理，三模型容灾
  • 非结构化数据 AI 原生理解
  • cronjob + webhook 双触发引擎

与 RPA 本质差异

  • 交互：拖拽编程 → 自然语言
  • 容错：选择器崩了就重录 → AI 自愈
  • 数据：只处理结构化 → 原生理解非结构化
  • 维护：改流程图 → 改一句话
  • 学习：2-4 周培训 → 零学习成本
